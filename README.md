# PrographyPhotoApp

### 프로젝트목적
Prography 동아리 제출용 과제

### Why SwiftUI?
유동적인 셀크기 형태의 스크롤뷰의 경우 과거 **UIKit**으로 작업한 경험이 있기 때문에(링크: [https://github.com/kirkim/ios-wanted-Picterest](https://github.com/kirkim/ios-wanted-Picterest))    
SwiftUI으로 작업을 결정하게 되었습니다.
